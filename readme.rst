Ce bot permet de jouer une partie de Tic-Tac-Toe sur Discord.

Commencez une partie avec !play, !play <nomJoueur> ou !playSolo
!move X pour placer votre "pion" sur la case X (1-9)
Si vous faites !play, le joueur contre qui vous jouerez sera la première personne à faire le !move après vous.
Si vous faites !play <nomJoueur>, la partie est entre vous et le joueur mentionné
Si vous faites !playSolo, la partie se jouera entre vous et le bot.

!stop pour arrêter la partie en cours.
Une seule partie peut être jouée à la fois.
Code un peu ignoble.
